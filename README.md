## Bogie Man

#### A web app for people to keep track of golf handicap and scores

###Working site:[Bogie Man](https://bogieman.herokuapp.com/)

### Features
* Handicap Tracker
* Score Tracker
* Visual Score representation


#### Technologies Used
* AngularJS
* Node.js
* Express.js
* PostgreSQL
* Bootstrap
* Chart.js
* Git
